# Workflow Obrigatório da IA

## Em TODA tarefa, a IA DEVE:

### Fase 1: Preparação
- [ ] Ler STATUS.md do módulo relevante
- [ ] Verificar "🐛 Bugs Conhecidos"
- [ ] Identificar tarefa prioritária
- [ ] Ler docs relacionados (README, ARCHITECTURE, DATABASE)

### Fase 2: Execução
- [ ] Seguir STYLE_GUIDE.md
- [ ] Seguir regras específicas do STATUS.md
- [ ] Testar localmente
- [ ] Documentar decisões importantes

### Fase 3: Finalização
- [ ] Atualizar STATUS.md:
  - Progresso (Completo/Em Progresso)
  - Data de conclusão
  - Decisões técnicas
  - Bugs novos (se houver)
- [ ] Commitar com mensagem clara
- [ ] Notificar usuário com resumo
